0安装

生成二维码的包不止一种，这里一个坑 qrcode 包，他也可以生成二维码，但是很难设置二维码大小，颜色等，强烈不建议安装这个。我已经掉坑里过。
那么推荐使用jquery.qrcode。

npm install jquery.qrcode -s

这里前提是需要安装jquery并且在webpack中配好，先说一下代码，以后会详细说明
